## 丨简介

原神多功能机器人，通过米游社接口查询uid的游戏信息，并提供WIKI查询和各种各样的好玩的功能。

该分支正在**开发中**，暂不可用于生产环境，可用版本在以下分支：

+ [nonebot2分支](https://github.com/CMHopeSunshine/LittlePaimon/tree/nonebot2)
